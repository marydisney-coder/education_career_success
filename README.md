##  Technologies Used:

* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

##  Dataset:

* **Education & Career Success Dataset**
* Key features include:

  * Academic performance (GPA, SAT scores)
  * Internships, projects, certifications
  * Soft skills and networking score
  * Job offers, salary, work-life balance
* **Target Variable:** Current Job Level

##  Data Processing:

* Handled missing values using imputation
* Encoded categorical variables
* Scaled numerical features
* Performed correlation analysis and visualizations

##  Model:

* **Logistic Regression (Classification)**
* Trained on standardized features
* Evaluated using accuracy score

##  Streamlit Web App:

The Streamlit app allows users to:

* Enter academic and career details
* Predict the **Current Job Level**
* View results instantly in an interactive interface
